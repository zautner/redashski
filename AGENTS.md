# Agent Instructions

### Always increase the version number in manifest.json for any changes
The version number is in the format X.Y.Z where X is major, Y is minor and Z is patch.
The volume and the character of changes determines which number to increment:
- Z is for small changes that do not affect functionality
- Y is for changes that affect functionality but do not break backward compatibility
- X is for changes that break backward compatibility
